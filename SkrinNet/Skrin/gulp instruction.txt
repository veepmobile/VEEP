1. Install Gulp globally with:

npm install gulp -g

2. Install Gulp into your project and save it as a development dependency with:
npm install gulp --save-dev

Add gulpfile.js

3. Install Gulp PostCSS
npm install --save-dev gulp-postcss


4. Install PostCSS plugins
npm install precss --save-dev
npm install postcss-nesting --save-dev


Using: 
gulp css or gulp css1
